# Next.js Cypress Starter

## ⚡️ Quick Start

```bash
yarn create next-app -e https://github.com/colbyfayock/next-cypress-starter
# or
npx create-next-app -e https://github.com/colbyfayock/next-cypress-starter
```

## 🚀 Getting Started

To start the project locally, run:

```bash
yarn dev
# or
npm run dev
```

The project should now be available at [http://localhost:3000](http://localhost:3000)!

## ⚙️ Running Tests

When using a `test:*` command, a new local server will be starter where Cypress will run the tests.

To run the Cypress tests with the UI, run:

```bash
yarn test:open
# or
npm run test:open
```

To run the Cypress tests only in the terminal, run:

```bash
yarn test:run
# or
npm run test:run
```
